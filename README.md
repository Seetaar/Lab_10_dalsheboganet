##  Описание проекта

Веб-приложение для предсказания одобрения ипотечного кредита на основе данных клиента с использованием машинного обучения.

### **ML Engineer**
**Сетало Елизавета**

**Выполненные задачи:**
-  Загрузка и анализ датасета `loan_data.csv`
-  Очистка данных (обработка пропусков и выбросов методом IQR)
-  Кодирование категориальных признаков (OneHotEncoder)
-  Масштабирование числовых признаков (StandardScaler)
-  Разделение данных на train/test выборки (80/20)
-  Обучение и сравнение моделей
-  Оценка качества моделей (ROC-AUC, Accuracy, Precision, Recall, F1-score)
-  Сохранение обученной модели и конфигурации
-  Написание тестов для ML pipeline

**Файлы:**
- `train_pipeline.py` - основной скрипт обучения
- `tests/test_pipeline.py` - тесты для ML компонентов
